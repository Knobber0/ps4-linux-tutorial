# Hyprland Optimizations

-**NOTE: I am making this for hyprland because i daily drive it both on my Playstation 4 Pro and laptop, if you know for other Desktop enviroments make a issue or a pull request on github**


| Hyprland tweaks                                      |                                                      				 	    |
|------------------------------------------------------|--------------------------------------------------------------------------------------------|
| decoration:blur:enabled = false 		       | Disables blur				     	      				 	    |
| decoration:shadow:enabled = false		       | Disables shadows		                      				  	    |
| misc:vfr = true 				       | Will lower amount of frames sent wen nothing happens 				            |
| AQ_NO_MODIFIERS=1                                    | Disables modifiers for DRM buffers.							    |
| RADV_PERFTEST=nggc,rt,sco		   	       | Enables Next Generation Geometry Culling,Rasterizer Toggle and Shader Cache Optimizations. |
